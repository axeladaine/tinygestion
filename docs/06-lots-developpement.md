# Lots de développement

## Lot 1 — Base simple et connexion sécurisée

Objectif :

* créer le backend ;
* créer le frontend ;
* mettre en place PostgreSQL ;
* créer l'authentification ;
* protéger les écrans ;
* créer le tableau de bord vide.

Fonctionnalités :

* connexion ;
* déconnexion ;
* changement de mot de passe ;
* création utilisateur initial ;
* rôles `PROPRIETAIRE` et `ASSISTANT` ;
* protection des routes ;
* journal de sécurité simple.

Livrables :

* écran de connexion ;
* écran tableau de bord ;
* backend sécurisé ;
* utilisateur de démonstration ;
* documentation de sécurité.

## Lot 2 — Logement, recettes et dépenses

Objectif :

* créer le logement ;
* saisir les recettes ;
* saisir les dépenses ;
* calculer les totaux mensuels.

Fonctionnalités :

* fiche logement ;
* liste des recettes ;
* formulaire recette ;
* liste des dépenses ;
* formulaire dépense ;
* calcul du montant retenu ;
* calcul du montant net reçu ;
* dashboard mensuel.

Livrables :

* écrans recettes ;
* écrans dépenses ;
* premier tableau de bord utile.

## Lot 3 — Justificatifs et biens amortissables

Objectif :

* joindre les factures ;
* suivre les achats durables ;
* calculer les amortissements estimatifs.

Fonctionnalités :

* upload de documents ;
* association document avec recette, dépense ou bien amortissable ;
* liste des biens amortissables ;
* formulaire bien amortissable ;
* calcul base amortissable ;
* calcul amortissement annuel ;
* calcul amortissement mensuel ;
* suivi facture manquante.

Livrables :

* écran documents ;
* écran biens amortissables ;
* calculs d'amortissement ;
* alertes justificatifs manquants.

## Lot 4 — Bilan annuel et checklists

Objectif :

* aider l'utilisateur à suivre ses obligations mensuelles et annuelles ;
* préparer un bilan annuel.

Fonctionnalités :

* checklist mensuelle ;
* checklist annuelle ;
* bilan annuel ;
* estimation résultat réel ;
* estimation micro-BIC indicative ;
* suivi amortissement utilisé ;
* suivi amortissement non utilisé.

Livrables :

* écran checklist ;
* écran bilan annuel ;
* synthèse annuelle.

## Lot 5 — Exports et dossier de contrôle

Objectif :

* permettre à l'utilisateur de sortir un dossier propre.

Fonctionnalités :

* export recettes CSV ;
* export dépenses CSV ;
* export biens amortissables CSV ;
* export impôts locaux CSV ;
* export bilan annuel PDF ;
* génération dossier de contrôle.

Livrables :

* exports ;
* PDF de synthèse ;
* dossier annuel structuré.
